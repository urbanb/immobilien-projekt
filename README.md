# 🏠 Immobilien-Projekt

Dieses Repository enthält alle Informationen, Skripte und Dokumentation für das Immobilien-Kaufprojekt.

## 📁 Ordnerstruktur

| Ordner | Beschreibung |
|--------|--------------|
| `01_Bank/` | Bankinformationen, Kreditangebote, Kontaktdaten |
| `02_Finanzierung/` | Finanzierungskonzepte, Kalkulationen, Budgetplanung |
| `03_Baumängelüberprüfung/` | Checklisten, Gutachten, Hinweise zur Bausubstanz |
| `04_Renditeanalyse/` | Renditeberechnungen, Cashflow-Analysen, Kennzahlen |
| `05_Immobiliensuche/` | Suchkriterien, gefundene Objekte, Automatisierungsskripte |
| `06_Steuern/` | Steuerliche Aspekte, AfA, Werbungskosten |
| `07_Dokumente/` | Allgemeine Dokumente, Verträge, Unterlagen |
| `08_Skripte/` | Automatisierungsskripte für Objektsuche |

## 🎯 Projekt-Ziel

Suche nach Wohnungen mit positiver Rendite auf dem Immobilienmarkt.
