# 🔄 Workflow für Inhaltsverarbeitung

## Automatische Kategorisierung

Wenn Videos/Texte zum Thema Immobilien eingehen:

1. **Transkribiere** das Video/Audio
2. **Analysiere** den Inhalt
3. **Ordne zu** in die passende Kategorie
4. **Prüfe auf Duplikate** → Doppelte Info wird übersprungen
5. **Bestätige** kurz: Was wurde wohin sortiert

## Kategorien-Mapping

| Thema | Zielordner |
|-------|-----------|
| Bank, Kredit, Zinsen | `01_Bank/` |
| Finanzierung, Budget, Eigenkapital | `02_Finanzierung/` |
| Baumängel, Gutachten, Bausubstanz | `03_Baumängelüberprüfung/` |
| Rendite, Cashflow, Mieteinnahmen | `04_Renditeanalyse/` |
| Objektsuche, Kriterien, Portale | `05_Immobiliensuche/` |
| Steuern, AfA, Werbungskosten | `06_Steuern/` |
| Verträge, allgemeine Unterlagen | `07_Dokumente/` |
| Automatisierung, Skripte | `08_Skripte/` |

## Duplikat-Prüfung

- Bereits vorhandene Infos werden nicht erneut gespeichert
- Neue Erkenntnisse werden ergänzt
- Bei Unsicherheit: User fragen

## Bestätigungs-Format

Nach jeder Verarbeitung melde ich kurz:
- ✅ Neuer Inhalt in `[Ordner]`
- 🔄 Keine neuen Infos (bereits bekannt)
- ❓ Unklar – bitte prüfen
